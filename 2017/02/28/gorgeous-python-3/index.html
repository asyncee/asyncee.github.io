Великолепный Python 3

Опубликовано 28 Февраля 2017

В этой стате я опишу возможности Python 3, которые облегчают жизнь разработчика. Доработок в новой версии языка действительно много, поэтому я опишу только наиболее заметные изменения. Я пишу код на Python 3 каждый день, и ежедневное использование этих возможностей приносит мне большое удовольствие. Итак, поехали!

print теперь функция, а не оператор

print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False) (документация).

Да, теперь print это функция, а не оператор, так что теперь необходимо использовать скобки: print("string").

unicode по−умолчанию

Документация по изменениям str vs unicode Документация по utf-8 кодировке по−умолчанию

Теперь все строки в Unicode по−умолчанию, кодировка, конечно−же, utf-8. Искренне рад этому изменению, так как теперь можно навсегда забыть литерал u"" и писать более чистый код.
